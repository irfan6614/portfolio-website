Fraud Detection · Imbalanced Data · Classification

Credit Card Fraud Detection System

A machine learning system designed to detect fraudulent credit card transactions — a high-stakes real-world problem where false negatives have serious financial consequences. The system uses a Random Forest ensemble classifier and handles the heavily imbalanced nature of fraud datasets (where genuine fraud makes up less than 1% of transactions) through SMOTE oversampling. Evaluated using precision, recall, F1, and ROC-AUC metrics.
